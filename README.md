# coc-kite-cmp

Kite complete for coc

## Install

`:CocInstall coc-kite-cmp`

## Keymaps

`nmap <silent> <C-l> <Plug>(coc-coc-kite-cmp-keymap)`

## Lists

`:CocList demo_list`
